Texto deletado

podemos marcar um texto como excluído para indicar que ele deve ser lido mas não considerado.

Nesse caso usamos a tag DEL

Texto inserido

podemos marcar um texto como inserido para dar um ênfase e indicar que ele foi adicionado depois.

Nesse caso usamos a tag INS

podemos fazer o mesmo com a tag u (não semantica)

Texto sobrescrito

Usamos um texto sobrescrito quando precisamos de algo assim: x20+10=240000

Nesse caso usamos a tag SUP.

Texto subescrito

Usamos um texto subescrito como em: H2O.

Nesse caso usamos a tag SUB.

Código Fonte - Pré formatação

O comando document.getElementById('teste') é escrito em linguagem JavaScript.

        
num = int(input('Digite um número'))
if num % 2 == 0:
    print (f'O número {num} é PAR')
else:
    print (f'O número {num} é ÍMPAR')
print ('Fim do programa')

Nesse caso usamos a Tag CODE para que ficasse em uma fonte na qual
todas as letras tem o mesmo espaçamento, o que torna o código mais legível e
utilizamos a tag PRE para que não perdesse a identação.
